## ✨ Overview

**Edit AI Pro** is a modern AI-powered photo editing web application that allows users to enhance, edit, and transform images using intelligent tools.

Built with a full-stack architecture, the platform provides a seamless experience with authentication, real-time editing capabilities, and a user-friendly dashboard.

## 🚀 How It Works

1. 👤 User signs up / logs in  
2. 🖼️ Uploads an image  
3. 🤖 AI processes the image  
4. ✨ Edited image is generated  
5. 💾 User saves and manages edits
